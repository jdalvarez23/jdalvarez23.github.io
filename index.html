Roblox Group Dashboard

A project created to facilitate the management of a gaming group in Roblox. It functions as a database, keeping track of promotions, users, and military branches in the group. It uses Firebase's Realtime Database to store all of the necessary information, Firebase Authentication to allow users to sign in to view the data, and the Roblox Web APIs to conduct certain group management operations.

JavaScript jQuery Firebase APIs Roblox APIs Node.js (Server)

Chemistry Quiz App

The Chemistry Quiz App is a mobile-responsive web application that was created at the request of a chemistry teacher who wants a more interactive way to teach ions to her students. Using Google's Spreadsheet API the app pulls the questions from a Google spreadsheet owned by the teacher, and is then displayed in a quiz format on the app. IndexedDB is used to store the questions the first time the user visits the web app.

Google APIs (Spreadsheet) Dexie.js (IndexedDB) JavaScript jQuery

TechDealz: An e-commerce site

TechDealz is a mobile-friendly e-commerce website. This website was created as an experimental project to further my knowledge of PHP and MySQL databases. Information is pulled from a MySQL database using PHP and SQL statements to display products, reviews, and more. Users are able to add items to their carts (either as a "Guest" or as a registered user). More features include an admin dashboard where admins can view, add, edit, or delete products, brands, categories, orders, and payments. Users are able to checkout through PayPal. Users also have a dashboard where they can edit their account settings, view their previous orders, set/edit their billing address, and view account information.

PHP MySQL JavaScript jQuery PayPal API
